# Materials

## 概要
ゲームで使用するマテリアルを配置します。

## フォルダ構成（推奨）
```
Materials/
├── Characters/      # キャラクターマテリアル
├── Environment/     # 環境マテリアル
├── Weapons/         # 武器マテリアル
├── Effects/         # エフェクトマテリアル
└── UI/              # UIマテリアル
```

## 命名規則
- `MAT_{カテゴリ}_{名前}` (例: `MAT_Character_PlayerArmor`)

## 注意事項
- シェーダーの最適化を意識
- テクスチャアトラスの活用
- マテリアルインスタンスの適切な使用
- モバイル対応の考慮
