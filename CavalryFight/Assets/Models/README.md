# Models

## 概要
3Dモデル（メッシュ）を配置します。

## フォルダ構成（推奨）
```
Models/
├── Characters/      # キャラクターモデル
├── Weapons/         # 武器モデル
├── Mounts/          # 騎乗動物モデル
├── Environment/     # 環境オブジェクトモデル
└── Props/           # 小道具モデル
```

## 命名規則
- `MDL_{カテゴリ}_{名前}` (例: `MDL_Character_Archer`, `MDL_Weapon_LongBow`)

## モデル作成ガイドライン
- **ポリゴン数**: ターゲットプラットフォームに応じて最適化
- **スケール**: Unity標準（1 unit = 1 meter）
- **ピボットポイント**: 適切な位置に設定
- **LOD**: 複雑なモデルにはLODを設定

## サポートフォーマット
- `.fbx` (推奨)
- `.obj`
- `.blend`

## 注意事項
- マテリアルは別ファイルとして管理
- UV展開を適切に
- 法線とタンジェントの設定
- リギング済みモデルのボーン構造を統一
